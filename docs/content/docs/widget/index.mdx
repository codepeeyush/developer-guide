The YourGPT chatbot widget is a fully customizable embeddable chat interface. This documentation covers the widget's internal architecture, configuration options, and extension APIs.

<Callout>
If you're new here, start with **Widget Types** to understand the three layout modes, then follow the suggested reading order below.
</Callout>
